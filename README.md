# MiniLibrary Project

## Project Description
This is a simple Java-based library management system created as part of the PGO course exercises. [cite_start]The project demonstrates the use of object-oriented programming basics, including class creation, constructors, methods, and logic handled by arrays and loops[cite: 212]. [cite_start]It allows adding books to a library, registering readers, and managing the process of borrowing and returning books[cite: 289].

## Class List
[cite_start]The project consists of the following classes located in the `src` folder[cite: 227]:

* [cite_start]**Book**: Represents a library book with properties like title, author, page count, and its current availability[cite: 238].
* [cite_start]**Reader**: Represents a library user, tracking their name, card number, and the number of books they have borrowed[cite: 256].
* [cite_start]**Library**: The main management class that stores an array of Book objects and contains logic for adding, searching, borrowing, and returning books[cite: 262, 286].
* [cite_start]**Main**: The entry point of the application which contains a test scenario to demonstrate the library's functionality[cite: 289].
